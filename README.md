# ManagePlus

**ManagePlus** is a real-world **Attendance & Wage Management System** designed for industrial and construction businesses to simplify workforce tracking.  
Built with **React.js (Web)**, **Android (Mobile)**, and **Supabase** as the backend, it enables seamless attendance management, payroll calculation, and reporting â€” all in one platform.
